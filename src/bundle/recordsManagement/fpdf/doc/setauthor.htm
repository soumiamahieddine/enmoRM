SetAuthor

SetAuthor(string author [, boolean isUTF8])

Description

Définit l'auteur du document.

Paramètres

author
Le nom de l'auteur.
isUTF8
Indique si la chaîne est encodée en ISO-8859-1 (false) ou en UTF-8 (true).
Valeur par défaut : false.

Voir

SetCreator(), SetKeywords(), SetSubject(), SetTitle().
